# Kursus Oversigt

## 7-Ugers Progression Visualiseret

```
Uge 1: Personlighed & Prompts 🎭
    │
    ├─ Lær: Prompt engineering, system prompts
    ├─ Byg: Basic AI chatbot med personlighed
    └─ Værktøj: n8n + OpenRouter
    
    ↓

Uge 2: Betingelser & Roller 🎯
    │
    ├─ Lær: IF/ELSE logic, Switch nodes, rolle-skift
    ├─ Byg: Multi-role agent (lærer, ven, quiz-master)
    └─ Værktøj: n8n conditions, Code nodes
    
    ↓

Uge 3: Hukommelse 🧠
    │
    ├─ Lær: Database design, samtale-historik
    ├─ Byg: Agent der husker brugere og samtaler
    └─ Værktøj: Supabase, PostgreSQL
    
    ↓

Uge 4: Værktøjer (HTTP/API/Scraping) 🛠
    │
    ├─ Lær: API integration, web scraping, tool calling
    ├─ Byg: Agent med adgang til vejr, nyheder, etc.
    └─ Værktøj: HTTP Request nodes, external APIs
    
    ↓

Uge 5: Multi-step Reasoning 🧩
    │
    ├─ Lær: Chain-of-Thought, ReAct pattern
    ├─ Byg: Agent der "tænker" trin-for-trin
    └─ Værktøj: Loop nodes, beslutnings-træer
    
    ↓

Uge 6: Multi-agent Samarbejde 🤝
    │
    ├─ Lær: Agent orchestration, specialisering
    ├─ Byg: Team af agenter der samarbejder
    └─ Værktøj: Workflow composition, delegation
    
    ↓

Uge 7: Deployment & Dokumentation 🚀
    │
    ├─ Lær: Production deployment, monitoring
    ├─ Byg: Deployed agent med Discord/Slack
    └─ Værktøj: Docker, cloud platforms, documentation
    
    ↓
    
🎓 FÆRDIG AGENT MED ALLE SUPERKRÆFTER! 🎉
```
