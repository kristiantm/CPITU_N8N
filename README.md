# CPITU_N8N - 7-Ugers Progressivt AI Agent Kursus

Velkommen til det progressive AI agent kursus! Over 7 uger bygger du din helt egen intelligente chatbot-agent, trin for trin.

## 🎯 Kursusmål

I slutningen af kurset har du bygget en færdig, funktionel AI-agent med:
- Personlighed og kontekst
- Hukommelse om brugere
- Adgang til eksterne værktøjer
- Multi-agent samarbejde
- Deployment i produktion

## 🛠 Teknologi Stack

- **Docker**: Containerisering af applikationen
- **n8n**: Workflow automation og agent orchestration
- **OpenRouter**: AI model integration (GPT, Claude, etc.)
- **Supabase**: Database og brugerdata
- **Discord/Slack**: Chat interface

## 📚 Kursusoversigt

### Uge 1: Personlighed & Prompts
Lær at give din agent en unik personlighed gennem prompts og grundlæggende AI-integration.

### Uge 2: Betingelser & Roller
Tilføj intelligent beslutningstagning og forskellige roller til din agent.

### Uge 3: Hukommelse
Gør din agent smart ved at gemme og hente brugerdata fra Supabase.

### Uge 4: Værktøjer (HTTP/API/Scraping)
Forbind din agent med omverdenen gennem APIs og web scraping.

### Uge 5: Multi-step Reasoning
Implementer kompleks beslutningslogik og multi-step tænkning.

### Uge 6: Multi-agent Samarbejde
Lad flere agenter arbejde sammen for at løse komplekse opgaver.

### Uge 7: Deployment & Dokumentation
Deploy din agent og dokumenter den professionelt.
