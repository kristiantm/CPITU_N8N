## Opgave 2: Skriv Dit System Prompt 📝

Baseret på din personlighedsprofil fra Opgave 1, skriv et komplet system prompt:

```
System Prompt:
_________________________________________________
_________________________________________________
_________________________________________________
_________________________________________________
_________________________________________________
```

**Husk at inkludere:**
- Hvem er agenten? (navn og rolle)
- Hvordan opfører agenten sig?
- Hvad er agentens særlige styrker?
- Er der nogle begrænsninger?

## Opgave 3: Test Din Agent 🧪

Implementer dit system prompt i din n8n workflow og test med følgende spørgsmål:

**Test Spørgsmål:**
1. "Hej! Hvad hedder du?"
   - Agentens svar: _____________________________

2. "Kan du hjælpe mig?"
   - Agentens svar: _____________________________

3. "Hvad kan du bedst?"
   - Agentens svar: _____________________________

4. "Fortæl mig en vittighed"
   - Agentens svar: _____________________________

5. "Hvad er meningen med livet?" (test begrænsninger)
   - Agentens svar: _____________________________

**Refleksion:**
- Hvad virkede godt? _____________________________
- Hvad skal forbedres? ___________________________
- Hvordan vil du justere dit prompt? ____________
